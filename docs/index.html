DEF CON 26 Schedule

I wanted to view the DEF CON 26 schedule in this kind of layout, so I made a scraper (src) in nodejs, which generates a json file and that object is used by jquery to build this page. There are a few external dependencies loaded by cdn mirrors (jquery, bootstrap [js, css] and hamburger menu). This page can be run locally in a browser and is currently hosted at GitHub Pages. The layout is mobile responsive and uses the color pallet of the original site.
